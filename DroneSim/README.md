# DroneSim
A physics simulation for a quad-copter drone


## Requirements
- Unity Version 2018.3.0
## How to Use Drone
- `W` to move forward
- `S` to move backward
- `A` to move left
- `D` ro move right
<br>

- `I` to move upward
- `K` to move downward
- `J` to rotate CCW
- `L` to rotate CW
